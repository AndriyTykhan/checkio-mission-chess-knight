Your friend loves chess very much but he isn't a strong player. You could help him by using your programming skills.
If this mission is too simple for you, try another one from this set - The shortest Knight's path.

Your task is to write a function which can find all checkerboard cells on which Knight could go for. The input data will be start cell and the amount of moves that Knight will make. Board is absolutely empty except this one figure - your Knight.
If one cell appears more than 1 time - do not append it more than once. You should return all possible cells in the list and sort them by the next rules: in alphabetical order (from 'a' to 'h') and in ascending order (from 'a1' to 'a8' and so on).

Input: Start cell, number of moves.

Output: List of all possible cells.

Example:

{% if interpreter.slug == "js-node" %} 
chessKnight('a1', 1) == ['b3', 'c2']
chessKnight('h8', 2) == ['d6', 'd8', 'e5', 'e7', 'f4', 'f7', 'f8', 'g5', 'g6', 'h4', 'h6', 'h8']
{% else %} 
chess_knight('a1', 1) == ['b3', 'c2']
chess_knight('h8', 2) == ['d6', 'd8', 'e5', 'e7', 'f4', 'f7', 'f8', 'g5', 'g6', 'h4', 'h6', 'h8']
{% endif %}

How it is used: For game analysis and bot (game AI) development.

Precondition:
1 <= the number of moves <= 2
